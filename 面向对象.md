#	面向对象

###	面向对象是什么

	面向对象是一种思想(一种开发的方式)

	 	面向对象师将具有共同特征的功能或者代码把他们归为一类(封装方法)
	把他们需要详细描述的特性挂在原型上的一种编程方式.

### 面向对象特征

	1.抽象性
	2.继承性
	3.封装性

###	面向对象中的new
	 new : 函数一元运算符。 专门用来运算的符号

        加了new不加括号依然调用，括号是拿来传参的。

        加了new之后构造中的this指向实例化对象

        默认return  this

        如果return 简单（基础）类型，那么返回值为实例化对象

        如果return 复合（复杂）类型，那么返回值为这个复合（复杂）类型。
